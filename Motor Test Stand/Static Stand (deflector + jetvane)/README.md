# Motor Test Stand — Ground Support Equipment

Printable PC-FR parts for the vertical load-cell thrust stand used to characterize 29 mm motors and to run the E16-4 jetvane material-erosion test (RQ3).

- `TS_base_plate.*` — staked ground plate.
- `TS_motor_tower.*` — vertical motor cradle + load-cell seat.
- `TS_loadcell_bracket.*` — bar load-cell mounting bracket.

Instrumentation: Wishiot 10 kg bar load cell + HX711, read by an Adafruit Metro M4 logging to microSD, calibrated with an Estes E16-4. Generated by `../3D parts/_generator/gen_rocket.py`.
